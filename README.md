# UPX-4-EcoMove
Trabalho UPX-4 - EcoMove

O EcoMove é um aplicativo desenvolvido em React Native com o objetivo de facilitar a mobilidade sustentável, oferecendo uma solução moderna, intuitiva e eficiente.

Antes de iniciar, certifique-se de ter instalado:

Node.js 22.19.0

(Versão necessária para compatibilidade com o projeto)

npm (já vem com o Node)

Ambiente configurado para React Native
(Android Studio, JDK, emulador ou dispositivo físico)

Use git clone no link: https://github.com/GuRosendo/UPX-4-EcoMove.git para clonar o repositório

Acesse a pasta do projeto: cd EcoMove

Instale as dependências: npm install

E rode com: npx expo start ou npx react-native run-android ou npx react-native run-ios ou npm run start

Este projeto é destinado ao uso acadêmico.
